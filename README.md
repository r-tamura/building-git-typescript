# kit

An implementation of jit ([Building Git](https://shop.jcoglan.com/building-git/)) in TypeScript

## Supported commands

```sh
# ./bin/kit <subcommand>
kit init
kit commit
kit add
kit status
kit branch
kit checkout
kit log
kit merge
```

## Development

```sh
# build
yarn watch

# test (unit & integ)
yarn test
yarn integ
```
